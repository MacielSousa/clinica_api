# Clínica API
 
## Esse projeto tem como objetivo criar uma API para atender a demanda de um laboratório/clínica de exames.
 
### Comandos para Executar esse projeto
 #### yarn
 #### docker-compose up --force-recreate
 #### yarn typeorm migration:run
